PropertyValueRequirements
*"next"*a function that returns anIteratorResultobjectThe returned object must conform to theIteratorResultinterface. If a previous call to the `next` method of anIteratorhas returned anIteratorResultobject whose *"done"* property is *true*, then all subsequent calls to the `next` method of that object should also return anIteratorResultobject whose *"done"* property is *true*. However, this requirement is not enforced.
