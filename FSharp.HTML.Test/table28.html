ComponentPurpose
LexicalEnvironmentIdentifies the Environment Record used to resolve identifier references made by code within this execution context.
VariableEnvironmentIdentifies the Environment Record that holds bindings created by |VariableStatement|s within this execution context.
PrivateEnvironmentIdentifies the PrivateEnvironment Record that holds Private Names created by |ClassElement|s in the nearest containing class. *null* if there is no containing class.
